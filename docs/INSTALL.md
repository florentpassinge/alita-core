# Install of alita

Alita was completly dockerize.

###### Informations of docker
1. PHP 7.4
2. NGNIX 1.14

###### List of url : 
* Database : 
    * Url : adminer.localhost
    * Login : same as `POSTGRES_USER` env
    * Password : same as `POSTGRES_PASSWORD`
    * Name of database : Alita
* Mailcatcher : 
    * Url : alita.localhost:8025
* Website : 
    * Url : alita.localhost
